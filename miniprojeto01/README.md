# Mini Projeto 01 - Python

## Descrição

Este projeto foi desenvolvido durante meus estudos de Python no curso da Data Science Academy (DSA).

O objetivo foi praticar conceitos básicos da linguagem Python e lógica de programação, criando uma tabela de vendas fictícias e trabalhando com os dados, para uma melhor análise das vendas.

## Tecnologias utilizadas

- Python 3

## Funcionalidades

- Importação das bibliotecas necessárias
- Geração de dados fictícios
- Vizualizaçãos dados
- Análise e processamento
- Exibição dos resultados em números e gráficos
